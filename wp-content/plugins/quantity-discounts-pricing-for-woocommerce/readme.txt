== Description ==

**Quantity Discounts & Pricing For WooCommerce**

This plugin allows you to create both product-based and cart-based discounts for your WooCommerce store. With this plugin, you can:

- Offer discounts on products based on quantity ranges.
- Set a fixed price per product quantity range.
- Create a pricing table to display discounts for various quantity ranges.
- Apply multiple discount rules to different products and categories.
- Save time by bulk applying rules to categories and user roles.

**Key Features:**

1. **Product-Based Discounts**: Set discounts for specific product quantities.
2. **Cart-Based Discounts**: Apply discounts based on the cart subtotal or quantity.
3. **User Role Pricing**: Define discounts specific to user roles.
4. **Dynamic Pricing Tables**: Display pricing details for different quantity ranges on product pages.
5. **Flexible Rule Creation**: Create multiple rules and apply them to specific products or categories.

This plugin ensures your customers are incentivized to purchase more while giving you full control over discount management.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/quantity-discounts-pricing` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **WooCommerce > Settings > Quantity Discounts** to configure the plugin.

== Frequently Asked Questions ==

= Can I set discounts for specific user roles? =
Yes, you can configure discounts based on user roles, making it perfect for wholesale pricing or member discounts.

= Is this plugin compatible with all WooCommerce themes? =
The plugin is designed to work seamlessly with most WooCommerce themes. For custom themes, some styling adjustments may be needed.

= Does this plugin support multi-language sites? =
Yes, it is compatible with popular translation plugins like WPML and Loco Translate.

= Can I apply rules to product categories? =
Absolutely. You can create rules and apply them to entire product categories, saving time on individual product configuration.
